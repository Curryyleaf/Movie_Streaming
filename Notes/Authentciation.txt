Yes, many third-party authentication services that use OAuth 2.0 offer free tiers or free usage for basic functionality. Here are a few popular options:

 1. Google
- Description: Allows users to log in with their Google accounts.
- Free Tier: Google provides a generous free tier for most applications, especially for basic authentication.

 2. Facebook
- Description: Users can authenticate using their Facebook accounts.
- Free Tier: Facebook's authentication is free, but there are limits based on the usage of the Graph API.

 3. GitHub
- Description: Users can sign in with their GitHub accounts, which is popular for developer-centric applications.
- Free Tier: Free for open-source and many private applications.

 4. Twitter
- Description: Authenticate users with their Twitter accounts.
- Free Tier: Basic usage is free, but there are limits on API calls.

 5. Auth0
- Description: A flexible, drop-in solution to add authentication and authorization services.
- Free Tier: Offers a free tier for up to 7,000 active users per month.

 6. Firebase Authentication
- Description: Provides authentication services that support email/password, social logins, and more.
- Free Tier: Free usage for up to 10,000 monthly active users.

 Summary
Most of these services provide free tiers suitable for development and small applications. Be sure to check the specific terms, as they may vary based on the volume of users and API calls. If your application scales, you might need to consider paid plans.
